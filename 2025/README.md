# 2025

## Setup

Install dependencies:

```bash
uv sync
```

## Run

Run all solutions:

```bash
python3 main.py
```


Run a specific day's solution:

```bash
python3 main.py 1
# Or
python3 1/task.py
```

## Formatting & Linting

Format code:

```bash
uv run ruff format .
```

Check and fix issues:

```bash
uv run ruff check --fix .
```
